# הנסיכה וצב הביצות — ספר אינטראקטיבי לרון

ספר ילדים אינטראקטיבי בעברית, מותאם לפעוט בן 2-3.
HTML/CSS/JS טהור — ללא תלויות. רץ אופליין מטאבלט אנדרואיד.

---

## הפעלה מהירה (פיתוח)

```bash
npx serve princess_turtle_book -l 3000 --no-clipboard
```

פתחו `http://localhost:3000` בדפדפן.

(התצורה גם מוגדרת ב-`.claude/launch.json` תחת `book`.)

---

## העברה לטאבלט אנדרואיד (אופליין)

יש שתי דרכים — ה**מומלצת היא PWA** (התקנה כמו אפליקציה).

### דרך א' — PWA (התקנה למסך הבית) ⭐

1. **העלו את התיקייה** `princess_turtle_book/` לשרת סטטי כלשהו ב-HTTPS:
   - GitHub Pages (חינם)
   - Netlify drag-and-drop (חינם)
   - Vercel
2. **בטאבלט**, פתחו את הקישור ב-Chrome.
3. תפריט (⋮) ← **"Install app"** או **"הוסף למסך הבית"**.
4. האייקון מופיע במסך הבית. לחיצה עליו → ספר במסך מלא, ללא דפדפן, עם cache אופליין מלא.

### דרך ב' — קבצים מקומיים (ללא שרת)

1. העתיקו את התיקייה `princess_turtle_book/` לטאבלט (USB / Drive).
2. התקינו אפליקציה כמו **HTTP Server** (Play Store).
3. הצביעו על התיקייה ופתחו את `index.html`.

> ⚠️ אין לפתוח `index.html` ישירות מ-`file://` — Service Worker לא יעבוד וקבצי האודיו עלולים להיכשל.

## חוויית רון

- **כריכה**: כפתור גדול "התחילו לקרוא" → נכנס למסך מלא, נועל את המסך מכבייה.
- **כל עמוד**: ההקלטה של אבא מתחילה אוטומטית.
- **מעבר עמוד**: סוויפ ימינה/שמאלה, או כפתורי החיצים הגדולים בתחתית.
- **לחיצה על האיור**: אנימציות עדינות לפי עמוד (ניצוצות / ניענוע / "..." בועות שתיקה).
- **עמוד 11** (הטוויסט): האיור מתחיל מטושטש עם "?" באמצע — לחיצה חושפת את ההפתעה עם קסם.
- **עמוד 12** (סיום): כפתור "🔄 מההתחלה" לחזרה.
- **רענון/יציאה**: הספר ימשיך מהעמוד האחרון בכניסה הבאה (כל עוד לא נלחץ "מההתחלה").

### כפתורים גלובליים
- 🔊 / ⏸ (פינה ימנית-עליונה): השמעה/עצירה של הקלטת העמוד הנוכחי.
- 🔊 / 🔇 (פינה שמאלית-עליונה): השתקה כללית. נשמרת ב-localStorage.

## החלפת הקלטה

ההקלטות יושבות ב-`audio/` בשם `cover.mp3` ו-`page_01.mp3` עד `page_12.mp3`.
להחליף הקלטה — פשוט החליפו את הקובץ באותו שם.

המרת M4A → MP3 (ffmpeg):
```bash
ffmpeg -y -i "Recordings/Page 1.m4a" -codec:a libmp3lame -b:a 128k -ar 44100 \
  princess_turtle_book/audio/page_01.mp3
```

> אחרי החלפת קובץ, אם הספר מותקן כ-PWA — צריך לעדכן את `CACHE_NAME` ב-`sw.js` (למשל מ-`v1` ל-`v2`) כדי שה-Service Worker יטען את הקובץ החדש.

## מבנה הקבצים

```
princess_turtle_book/
├── index.html        ← הספר עצמו (CSS+JS inline)
├── manifest.json     ← הגדרות PWA
├── sw.js             ← Service Worker (cache אופליין)
├── icon-192.png      ← אייקון 192×192
├── icon-512.png      ← אייקון 512×512
├── README.md         ← הקובץ הזה
├── images/           ← 13 איורים (00_cover + 01..12)
└── audio/            ← 13 הקלטות MP3 (cover + page_01..page_12)
```

## פתרון בעיות

| בעיה | פתרון |
|------|------|
| אין צליל בכניסה לעמוד 1 | דפדפנים חוסמים autoplay לפני אינטראקציה. הכפתור "התחילו לקרוא" סופר כאינטראקציה — אחריו autoplay עובד. |
| המסך נכבה תוך כדי קריאה | Wake Lock נכשל (אנדרואיד ישן). הגדירו ידנית timeout מסך ארוך בהגדרות הטאבלט. |
| הקלטה ישנה ממשיכה לנגן אחרי החלפה | עדכנו `CACHE_NAME` ב-`sw.js` (ראו "החלפת הקלטה"). |
| הספר נפתח מאמצע במקום מההתחלה | זה בכוונה — Resume אוטומטי. לחצו "🔄 מההתחלה" בעמוד 12 כדי לאפס. |
| יציאה ממסך מלא בטעות | בכריכה הבאה ה-Resume ימשיך מאותו עמוד. |
